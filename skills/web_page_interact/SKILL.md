# WebPageInteract Skill

## 功能
在当前打开的网页中，根据给定目标找到并交互指定元素。支持滚动查找、点击、双击等操作。

## 参数
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `goal` | str | 是 | 交互目标的自然语言描述 |
| `hints` | list[str] | 否 | 帮助 VLM 识别目标的提示信息 |

## 执行步骤
1. 观察当前页面内容
2. 根据目标描述和提示信息定位目标元素
3. 如果目标不在可视区域，滚动页面查找
4. 执行交互操作（点击/双击）
5. 确认操作结果后报告完成

## 适用场景
- 在文件列表中找到并打开特定文件
- 点击页面上的特定按钮或链接
- 在表单中选择特定选项
