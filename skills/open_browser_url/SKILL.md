# OpenBrowserUrl Skill

## 功能
使用系统默认浏览器打开指定 URL。通过 `Win+R` 运行对话框输入网址的方式启动。

## 参数
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `url` | str | 是 | 要打开的完整 URL |

## 执行步骤
1. 按 `Win+R` 打开运行对话框
2. 输入目标 URL
3. 按回车键确认
4. 等待浏览器窗口出现
5. 确认页面开始加载后报告完成

## 预期结果
浏览器成功打开并开始加载目标页面。

## 适用场景
- 打开任意网页
- 作为 Web 工作流的第一步
