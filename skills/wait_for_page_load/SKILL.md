# WaitForPageLoad Skill

## 功能
等待当前网页加载完成。通过固定等待时间或截图对比来判断页面是否稳定。

## 参数
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `timeout` | int | 否 | 最大等待秒数 (默认 5) |
| `description` | str | 否 | 等待的页面描述 |

## 执行步骤
1. 等待指定时间让页面资源加载
2. 截图观察页面是否已加载完成
3. 报告完成

## 适用场景
- 打开新页面后等待加载
- 执行操作后等待页面响应
- 页面重定向后等待新页面出现
